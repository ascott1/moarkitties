# moarkitties
> replace all the images on a page with cat gifs

![](https://raw.github.com/trolljs/moarkitties/master/catoogle.png)

## Install

```html
<script src="dist/moarkitties.min.js"></script>
```

or via browserify:

```sh
$ npm install --save moarkitties
```
